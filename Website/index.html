Infected

Infected is a Death-Match Multiplayer Mod where you can play with your friends as Zombies against Humans. Inside of a beautiful mansion, protect your home for a given duration in time chosen by the Host Server as it is attacked by Player Zombies.

Zombies

Zombies have spawning points inside of the forest. The goal of a Zombie is to infect all of the Humans and turn them into Zombies as you kill them with your sword. Though Zombies do not have Ammo, they have a huge Melee advantage when coming into close combat. They also have a huge advantage in their speed.

Weapon(s)

 Katana Sword

laLives

 Unlimited

Speed

 Enhanced

 

Humans

Humans spawn inside of the mansion. Their advantage is when they spot a Zombie from a long distance. Any melee combat is nearly impossible since humans don't inflict much damage except with the limited ammo that they have equiped. Health and ammo can be replentished in the dark forest where Zombies spawn when killed.

Weapon(s)

 Crossbow, Shotgun, Dagger, Molotov

Lives

 Only one before turning into a Zombie

Speed

 Normal

 

Installation Guide


Download Link


*You must have a steam account with Half Life 2: DeathMatch (doesn't have to be installed) and Source SDK / Source SDK Base 2007 Installed. Also make sure you have run source sdk base 2007 atleast once. Both these programs are available under Tools in Steam.

Extract Infected folder in zip file to <Steam Directory>\steamapps\sourcemods\

Restart Steam and you will see Infected under your games list.

Start the game and either create a server or connect to a friends server. If you create a server make sure to enable Teamplay, Flashlight, and set a timelimit in the game tab

Note: Currently only local LAN games work.  If you wish to connect,
you must open the console by hitting the tilde and typing ‘connect
ip/hostname’. You may have to enable the console under Options, Keyboard, Advanced
